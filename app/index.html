Introduction

This is a two part challenge illustrated with Section 1 and Section 2.
You are free to use any tools and resources you see fit as long as you follow the few guidelines for each section.

Each section has a set of problems with pre-defined steps you need to implement.

  • Pre-requisite

    Before diving in, you will need to be familiar with the concept of Page Objects (PO).
    You don't need to be an expert in Page Object Model (POM) since the implementation in itself is slightly different at AlayaCare but understanding the idea is necessary.

    Inside cypress/objects you will find two files section-1.js and section-2.js containing skeleton POs for each section.

    ‚ö†Ô∏è Build on top of them in order to create more complete POs.

  • Section 1

    In this section, you will be facing basic test assertions and interactions:

    1. How to select DOM elements?
    2. How to interact with DOM elements (click, type, select multiple...)?
    3. How to do basic assertion checks?
  • Section 2

    In this section, you will be facing various API interactions:

    1. How to avoid flaky tests in the event of slow network requests?
    2. How to handle browsers' API when opening a new tab?
    3. How to handle browsers' API when downloading a file?
